# Banner Generator App

Приложение для создания баннеров с поддержкой сохранения настроек в локальное хранилище IndexedDB.

## Требования

Перед запуском приложения вам потребуется установить:

1. Node.js - скачайте и установите с официального сайта: [https://nodejs.org/](https://nodejs.org/)

## Как запустить приложение

1. После установки Node.js, установите зависимости:

```bash
# Если вы используете npm (устанавливается вместе с Node.js)
npm install

# Если вы установили pnpm
pnpm install

# Если вы установили Yarn
yarn
```

2. Запустите сервер разработки:

```bash
# Если вы используете npm
npm run dev

# Если вы используете pnpm
pnpm dev

# Если вы используете Yarn
yarn dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

## Альтернативный способ запуска

Если у вас установлен Docker, вы можете запустить приложение в контейнере:

```bash
# Создание Docker образа
docker build -t banner-generator .

# Запуск контейнера
docker run -p 3000:3000 banner-generator
```

## Функциональность

- Генерация баннеров для App Store
- Настройка цветов, текста и позиционирования
- Загрузка скриншотов
- Многоязычная поддержка
- Сохранение настроек в IndexedDB для последующего использования
- Экспорт баннеров в формате PNG

## Технические детали

- Next.js
- React
- IndexedDB для локального хранения
- html2canvas для рендеринга
- JSZip для создания архивов с баннерами # Trigger deploy
